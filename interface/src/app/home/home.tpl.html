ADAGE: Home

The ADAGE web server allows users to analyze signatures from certain types of unsupervised machine learning models. If you use the ADAGE server, please cite:
Tan J, Huyck M, Hu D, Zelaya RA, Hogan DA, Greene CS. ADAGE signature analysis: differential expression analysis with data-defined gene sets BMC Bioinformatics. 2017.
Please also cite the relevant paper for the model that you use, which is shown after a model is selected.
